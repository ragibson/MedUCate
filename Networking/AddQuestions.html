Welcome to the Excel Question Creation tutorial. This tutorial is comprised of three

your questionset.

Task 1: Preparing your computer

The first task to get started is to create the folder that our tool will use to store the file for you to upload.

Start by navigating to your "C" drive in your computer's file explorer.




Then, create a new folder my right clicking inside the "C" drive and mousing to "new" then "folder"




Now, rename this folder to "CSVToAdd"




Congratulations, you have now created the folder needed to get started!

Task 2: Downloading and using our tool

The next task on our list will be downloading and using the tool.


Start by clicking on the link to our tool at the top of the webpage.




Now, open up the file you downloaded to get started.




To add more questions to the list, click the "Add" button




To remove questions, click the "Remove" button




Type your questions into the blanks as specified in the examples. When you are done, click the "Add Question" button




If you have used our tool before, make sure to click "Yes" when it asks if you would like to replace your old file




Close out of excel. Congratulations, you are now finished using the Excel Question Creation Tool!

Task 3: Uploading formatted questions

The final task is to upload your new file to our server.


On our webpage, navigate to the Upload Center and click on "Choose File"




Navigate to the "C" drive




Now open the "CSVToAdd" Folder we created earlier.




Here you will find the correctly formatted question set that you created, double click on the "QuestionsToAdd.csv" file to select it.




Click "Open" once the file is selected.




Now, hit the "Upload Now" button in the Upload Center.




You should be brought to a page saying "The file QuestionsToAdd.csv has been uploaded".

Congratulations! You are now done with all tasks. We have received your question set and will add it to the game as soon as we can.
